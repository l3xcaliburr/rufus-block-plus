# Privacy Policy for Amazon Rufus Blocker

**Last updated:** April 2, 2026

## Data Collection

Amazon Rufus Blocker does not collect, store, transmit, or share any user data of any kind. The extension runs entirely on your local device using a CSS stylesheet and a JavaScript content script that execute only on Amazon pages.

## What the Extension Does

The extension injects a local CSS file and a local JavaScript file into Amazon pages to remove the Rufus AI chatbot panel and reclaim the screen space it occupies. The cart functionality is preserved by repositioning the existing cart widget as a toggleable overlay.

## What the Extension Does Not Do

- Does not collect personal information
- Does not track browsing history
- Does not use analytics, telemetry, or cookies
- Does not make any network requests
- Does not store any data locally or remotely
- Does not communicate with any external servers
- Does not sell or transfer any data to third parties
- Does not use or transfer data for purposes unrelated to its single purpose
- Does not use or transfer data to determine creditworthiness or for lending purposes

## Permissions

The extension requires host permissions for Amazon domains solely to inject CSS and JavaScript that remove the Rufus AI chatbot panel. These permissions are not used for any other purpose. The full list of matched domains is limited to Amazon storefronts including amazon.com, amazon.co.uk, amazon.ca, amazon.de, amazon.fr, amazon.it, amazon.es, amazon.co.jp, amazon.com.au, amazon.in, and amazon.com.br.

## Changes to This Policy

If this privacy policy is ever updated, the changes will be reflected in this document with an updated revision date.
